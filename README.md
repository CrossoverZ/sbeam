# sbeam
## web项目:sbeam游戏商城
### 项目构成：
IDE: IntelliJ IDEA

项目框架: 

1. 前端(商城,前后端不分离)：thymeleaf
2. 前端（管理台,前后端分离）：Vue
3. 后台：SpringBoot + mybatis-plus + shiro

项目管理：Maven

数据库：MySQL

数据库生成文件：mysql/sbeam.sql

管理后台前端代码： ./web
