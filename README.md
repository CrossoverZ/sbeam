# sbeam
## 毕业项目:sbeam游戏商城
### 项目构成：
IDE: IntelliJ IDEA

项目框架：Spring + Spring MVC + Mybatis

项目管理：Maven

数据库：MySQL

数据库生成文件：mysql/sbeam.sql
